# Bitcoin UI Sales Dashboard Demo

Steps on getting started
* Clone/Download Zip
* Open with Visual Studio 
* Update Bunifu Framework DLLs

Bunifu Framework Controls used in this project

* BunifuFlatButton
* Bunifu DataViz
* BunifuCustomLabel
* BunifuCards
* BunifuCustomDataGrid
* BunifuElipse
* BunifuDragControl
* BunifuSeparator
